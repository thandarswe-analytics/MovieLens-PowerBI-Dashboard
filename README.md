# MovieLens Business Intelligence Dashboard

## Project Overview

This project analyzes movie trends, user demographics, genre preferences, and rating behavior using the MovieLens dataset. An interactive Power BI dashboard was developed to generate business insights through data visualization and analytics.

## Tools Used

- Power BI
- DAX
- Power Query
- Data Visualization

## Dataset

- 3,883 Movies
- 301 Genres
- 6,040 Users

## Dashboard Features

- KPI Cards
- Demographic Analysis
- Genre Analysis
- Rating Trends
- Interactive Filters
- Dashboard Navigation

## Key Insights

- Identified top-rated genres.
- Analyzed demographic engagement patterns.
- Evaluated rating trends across decades.
- Studied user behavior by age and occupation.
